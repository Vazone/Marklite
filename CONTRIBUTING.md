# Contributing to MarkLite

Thanks for your interest in MarkLite.

MarkLite is maintained by Vazone. The repository is public, but changes should be submitted through pull requests. Please do not push directly to the main branch unless you are the maintainer.

## Pull Request Flow

1. Fork the repository.
2. Create a feature branch.
3. Run the checks:

```bash
npm install
npm run check
npm run build
```

4. Open a pull request against `main`.
5. Wait for review and approval from Vazone.

## Notes

- Keep changes focused.
- Do not commit build output such as `dist/`, `src-tauri/target/`, installers, logs, or local cache files.
- If you believe anything in the project unintentionally infringes your rights, please open an issue or contact Vazone through the repository.
